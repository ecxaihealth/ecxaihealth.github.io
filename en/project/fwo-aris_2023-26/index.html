Enriched conversational XAI methods for healthcare

FWO-ARIS project (2023-2026)

Jan 1, 2023

Main host institution: KU Leuven
Supervisor: Katrien Verbert (Informatics Section, KU Leuven)

Foreign host institution: University of Maribor
Supervisor: Gregor Štiglic (FHS, University of Maribor)

Project reference in the Slovenian Current Research Information System (SICRIS): here

Summary
The field of artificial intelligence is increasingly being applied in the healthcare sector, where the latest techniques rely on machine learning. The challenge of understanding these models remains. The use of Explainable Artificial Intelligence (XAI) can make such models more understandable to the end-user. In this project, we aim to use Explainable Artificial Intelligence to enable end-users to explain and justify the predictions and recommendations obtained. In doing so, we will focus primarily on more general, broad, and flexible explanations.

This is to enable healthcare professionals to understand the reasoning behind the model used. This would have an impact on both acceptance and confidence in the model. In this project, we will investigate the effectiveness of using conversational explanatory interfaces that are able to explain model decisions in dialogue with the end-user. Finally, we want to investigate how explanations can be tailored to the end-user.

The project will be based on an empirical approach in which we will investigate how users interact with the explanatory interface and assess the impact of explanatory methods on model understanding and user confidence. The data collection will involve ten healthcare centers in Slovenia, where we will apply the XAI methods developed in this project.
